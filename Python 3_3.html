  • interpreter 和 compiler 的差異
    • interpreter
      • 執行時，一行接著一行依序將程式碼轉換成機器碼執行
      • 執行時期才做轉換，導致執行效率較慢
      • e.g. Python, Bash
    • compiler
      • 一口氣將全部程式碼轉換成機器碼
      • 執行速度較快
      • e.g. C, C++

主要的變數型別

  • None
    • 空的，沒有任何內容
  • boolean
    • 布林
    • e.g. True, False
  • integer
    • 整數
    • e.g. 1, 2
  • float
    • 浮點數
    • e.g. 1.2, 1.0
  • string
    • 字串
    • e.g. 'hello', '', ""
  • list
    • 列表
    • e.g. [], [1, [2, 3], 'hello', 5.7]
  • tuple
    • 元組
    • e.g. (), (1, [2, 3], 'hello', 5.7)
  • dictionary
    • 字典；key-value 的配對集合
    • e.g. {}, {'aaa':'a', 2:'333', 'z':[2, 3]}
  • 重要差同
    • string
      • 單引號、雙引號皆可，不論用哪一個開始字串，只能用同一個結束字串
    • integer vs. float
      • 有小數點的是 float；沒有的是 integer
      • integer 之間運算，只會產生 integer。遇到小數點部分會捨去。
    • list vs. tuple
      • list 宣告後可改變內容；tuple 則不行
    • list, tuple and dictionary
      • 都可以用 key 來取得內容
      • >>> a = [1, 2, 3]
        >>> a[0]
        1
        >>> a[2]
        3
        >>> b = (2, 3, 4)
        >>> b[1]
        3
        >>> c = {'a':123, 234:'def'}
        >>> c['a']
        123
        >>> c[234]
        'def'
    • string, list, tuple
      • 都可以使用 slice
      • >>> "abcdefg"[1:3]
        'bc'
        >>> "abcdefg"[5:18]
        'fg'
        >>> "abcdefg"[-5:18]
        'cdefg'
        >>> "abcdefg"[:]
        'abcdefg'
        >>> "abcdefg"[2:]
        'cdefg'
        >>> "abcdefg"[:3]
        'abc'
        >>> "abcdefg"[1:5:2]
        'bd'
        >>> "abcdefg"[-1:0:-2]
        'gec'

Built-in Functions: int(), raw_input(), input()

https://docs.python.org/2/library/functions.html#int
# 字串或浮點數轉整數
>>> int('645')
645
>>> int(64.5)
64
https://docs.python.org/2/library/functions.html#raw_input
>>> s = raw_input('--> ')
--> Monty Python's Flying Circus
>>> s
"Monty Python's Flying Circus"
  • raw_input() vs. input()
    • raw_input() 單純回傳字串
    • input() 則試圖運算內容再回傳
    • >>> print(input('?'))
      ?1+2*3
      7
      # 所以推薦使用 raw_input()
