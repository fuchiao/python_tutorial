  • 要注意的事項
    • indent
      • python 藉由 indent 來界定架構內容的範圍
      • tab 和 space 不能混用，必須固定
      • 建議固定使用四個空白
      • 可以利用編輯器設定，將所有輸入的 tab 切換成四個空白
      • e.g.
      • # test1.py
        if score > 60:
            print('qualified')
        print('ByeBye')
      • # test2.py
        if score > 60:
            print('qualified')
            print('ByeBye')
    • 沒有 statement 的 if block，會導致錯誤訊息
      • 可以放入 pass，代表不做任何事
      • if temperature > 33:
            print('drink beer')
        elif temperature > 20:
            print('watch tv')
        elif temperature > 15:
            print('read comic')
        elif temperature > 5:
            pass
        else:
            print('call 911')
